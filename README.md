E-commerce Product Classification
Overview

This project uses a Convolutional Neural Network (CNN) built with TensorFlow/Keras to classify e-commerce product images into their respective product categories.

Model Details
Framework: TensorFlow / Keras
Architecture: CNN
Input Image Size: 224 × 224 pixels
Output: Product Category Prediction
Repository Contents
notebooks/ – Kaggle notebook used for training and evaluation
requirements.txt – Project dependencies
README.md – Project documentation
Trained Model

The trained model is hosted on Google Drive because it exceeds GitHub's file size limit.

Download the model here:

https://drive.google.com/file/d/1eRYuCBOPyV7Y22zIkWi9uSLO_hRGfLhH/view?usp=sharing

After downloading, place the model file in:

models/cnn_model.keras

Installation

Install the required packages:

pip install -r requirements.txt

Usage
Download the trained model from the Google Drive link above.
Place the model in the models folder.
Run the notebook or deployment application.
Upload a product image and obtain a category prediction.
